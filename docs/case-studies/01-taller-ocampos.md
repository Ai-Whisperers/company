# Case Study: Taller Ocampos — Auto Repair Shop Management System

> Flagship example of a vertical SaaS built end-to-end.
> Last updated: 2026-06-15

---

## 🏢 The Client

**Taller Ocampos** — an auto repair shop (taller mecánico) management system. The shop needed to digitize client management, vehicle tracking, work orders, invoicing, and inventory in a single system that could be operated by non-technical staff.

---

## 🎯 The Problem

- Manual paperwork for every work order
- No real-time visibility into shop status for owners
- Disconnected inventory tracking
- Slow invoicing and payment processing
- No client history or vehicle service records

---

## 💡 The Solution

A complete management system built across two apps:

### Frontend (Next.js 14 + React 18)
- Modern UI with Tailwind CSS + Radix UI
- Responsive (desktop, tablet, mobile)
- JWT authentication
- Real-time dashboard with WebSocket updates
- Client & vehicle management
- Work order tracking
- Invoicing & payment system
- Inventory management
- Full test coverage (Jest + Playwright)

### Backend (Express + PostgreSQL)
- RESTful API
- Real-time WebSocket for live updates
- JWT auth
- Inventory management
- Payment processing
- Client/vehicle/work order/invoice data model

---

## 📊 The Results

- **Production-ready:** Vercel-deployed frontend + Render-deployed backend
- **5-minute deploy:** Quick-start guide in README
- **Real-time:** Owner sees shop status as work happens
- **Mobile-first:** Mechanics can update work orders from the shop floor
- **Full-stack coverage:** Frontend + Backend + Tests + Docs
