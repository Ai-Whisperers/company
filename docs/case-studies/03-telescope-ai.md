# Case Study: telescope-ai — MCP Server for Real Telescope Control

> The proof point that we don't just write about MCP — we ship hardware-control MCP servers that work in production.
> Last updated: 2026-06-15

## 🎯 The Project

**telescope-ai** is an AI-powered control system for Celestron NexStar telescope mounts. It exposes a Model Context Protocol (MCP) server that allows AI agents to control real telescope hardware, including:
- Satellite tracking
- Night scheduler
- Plate solving
- Object classification

Built to demonstrate the power of MCP for hardware integration, telescope-ai is a working example of how AI agents can interface with physical devices — not just software.

---

## 🛰️ Capabilities

### Hardware Control
- Connect to Celestron NexStar telescope mounts (USB/serial)
- Slew to celestial objects
- Track satellites (ISS, Hubble, etc.)
- Plate solving (astrometric calibration from images)
- Object classification (star, galaxy, nebula, etc.)

### AI Features
- Natural language commands ("point at Mars")
- Predictive scheduling (when to observe what)
- Auto-classification of captured images
- Session logging and analysis

### MCP Server
- Standard MCP protocol interface
- Any MCP-compatible AI agent can control the telescope
- Python implementation
- Test coverage included
