## 🎯 Quick Start Guide

### For Job Applications

1. **Select the right resume**:
   - Use `01-CV-Resume/Ivan_Weiss_Resume_ATS_Optimized.txt` for online applications
   - Use `01-CV-Resume/Ivan_Weiss_CV_Comprehensive.md` for detailed submissions

2. **Choose a cover letter**:
   - Open `02-Job-Application-Materials/Cover_Letter_Templates.md`
   - Select template matching the role (Software Engineer, Data Engineer, AI/ML, Power Platform, General)
   - Customize with company-specific details

3. **Prepare for interviews**:
   - Review `07-Interview-Prep/Interview_Preparation_Guide.md`
   - Practice STAR method responses

4. Review featured post templates for content ideas

2. Copy the README.md content

### Customization Instructions

**CV/Resume:**
1. Update the `[Date]` placeholder with current date
2. Adjust sections based on job requirements
3. Emphasize relevant projects for specific roles
4. Keep contact information current

**Cover Letters:**
1. Replace all `[bracketed placeholders]` with specific information
2. Research company and customize paragraph about "Why this company"
3. Match tone to company culture (formal vs. casual)
4. Keep to one page maximum

**LinkedIn:**
1. Choose summary version based on career goals
2. Update headline to match target roles
3. Customize featured section with recent achievements
4. Add skills relevant to target positions
